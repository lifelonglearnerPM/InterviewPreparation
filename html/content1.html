Keywords

Keywords are reserved words in a programming language that have a special meaning to the compiler or interpreter.

// Example keywords in JavaScript
if, else, while, for, function, return, var, let, const

Identifiers

Identifiers are names given to elements in a program such as variables, functions, arrays, etc.

// Example identifiers in JavaScript
let myVariable = 10;
function myFunction() {
    // Function code
}

Variables

Variables are containers for storing data values.

// Example variables in JavaScript
let num = 5;
let name = 'John';

Data Types

Data types specify the type of data that a variable can hold.

// Example data types in JavaScript
let number = 10; // Number
let text = 'Hello'; // String
let isTrue = true; // Boolean
