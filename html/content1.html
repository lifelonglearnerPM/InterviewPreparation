Keywords

Keywords are reserved words in a programming language that have a special meaning to the compiler or interpreter.

// Example keywords in JavaScript
if, else, while, for, function, return, var, let, const

Identifiers

Identifiers are names given to elements in a program such as variables, functions, arrays, etc.

// Example identifiers in JavaScript
let myVariable = 10;
function myFunction() {
    // Function code
}

Variables

Variables are containers for storing data values. They are essential for any programming language.

Example Code Block

// This is a code block example in JavaScript
let x = 10;
let y = 20;
let sum = x + y;
console.log(sum); // Outputs 30

Datatypes

Data types specify the type of data that a variable can hold. Common data types include integers, floating-point numbers, characters, strings, and arrays.

Example Table

Type Description
int Integer type
float Floating point type
string String type
